# Todo list app

A todo list app

### General steps to follow when creating a project

- Thinking about the UI
- Functionality
  - Add a TODO
  - Delete a TODO
  - Check tasks
  - Total items count
- Data
  - tasks - an array
  - task - {done, text, id}
- Functions
  - addTodo
  - deleteTodo
  - checkTodo
  - renderTodoList
  - showNotification
